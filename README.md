# Mouse Beat

Mouse Beat is a small browser rhythm game. Click the targets as they appear and keep your combo going.

## Play locally

On Windows, double-click:

```bat
start_mouse_beat.bat
```

Then open the browser page that appears.

## Publish with GitHub Pages

This folder is ready to publish as a static site. Push the contents of this folder to a GitHub repository, then enable GitHub Pages from the repository root.

Required files and folders:

- `index.html`
- `styles.css`
- `game.js`
- `dog3d.js`
- `assets/`
- `vendor/`
